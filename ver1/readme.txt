このフォルダの中身は以下の通りです。

①IGAシステムで体内音/自己聴取音のパラメーターを決定する(python3 main.py で動く)
 main.py ... 本体
 GA.py ... 遺伝的アルゴリズムで新しいパラメーターを出す部分
 gen_sound.py ... パラメーターから音を生成する部分
 my.kv ... アプリとして触れるようにするための部分

②IGAシステムで作られたパラメーターを元に任意の録音音声を体内音or自己聴取音化する(python3 main2.py で動く)
 main2.py ... 本体
 convert_sound.py ... 選択したパラメーターのファイルを元に音源を変換する部分
 conversion.kv ... アプリとして触れるようにするための部分

③その他のファイル・フォルダ
 melcp.py ... 2つの音源のメルケプストラム歪みを計算する
 normalize.py ... 音を正規化する(①、②にも組み込まれています)
 to_monoral.py ... ステレオからモノラルにする
 __pycache__ ... コンパイルしたときに作られるファイルであるとちょっと処理が早くなる(らしい)
 font ... アプリで使われるフォントが入っている
 requirements.txt ... ①と②を動かすのに必要なライブラリが書かれている


また、IGAシステムver1とver2では以下の違いがあります。
ver1 ... 体内音と自己聴取音を個別に推定する
ver2 ... 体内音を推定してからそのパラメーターを引き継いで自己聴取音を推定する
